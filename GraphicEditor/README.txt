Особенности кода:
-Весь код написан в соответсвии c ООП
-Использованы разные модули библиотеки PYQT5, которые глубоко мы не изучали на уроках
-Использованы различные фишки python
-Оптимизированность кода - Удаление ненужных объектов при рисовании
Смысл проекта:
Сделать удобную красивую графический редактор, которую не стыдно показать публике


Если хотите просмотреть дизайн программы, можете открыть папку design
